## Competitor Analysis

### Competitor 1: "Inteligência Artificial na Perícia" — Edilson Aguiais (IBCAPPA)

| Attribute | Value |
|-----------|-------|
| Publication Year | 2024-2025 (v2 e v3 lançadas) |
| Platform | IBCAPPA + Clube de Autores + Google Play |
| Bestseller Rank | Referência no nicho (sem ranking Amazon direto) |
| Number of Reviews | Positivas em plataforma própria |
| Rating | ~4.5 stars (estimado via comentários) |
| Price | R$ 49–97 (ebook/impresso) |
| Format | Ebook + Impresso + Curso |

**Positioning:** "O mapa prático e visionário para peritos transformarem sua carreira com IA"

**Promise:** "Ganhar tempo, precisão e autoridade usando IA na perícia"

**Strengths:**
- Autor é maior referência nacional em perícia + IA (10+ anos, milhares de alunos)
- Ecossistema completo: livro + curso + agentes IA (Rebecca, Sofia, Alceu)
- Método CLARA proprietário para prompts periciais
- Múltiplas edições (v2, v3) — iteração constante
- Comunidade ativa de peritos

**Weaknesses (from reviews/análise):**
- Cobre múltiplas ferramentas (ChatGPT, Gemini, Claude, Perplexity) de forma genérica — raso em cada uma
- Não aprofunda nas vantagens específicas do Claude para análise documental
- Foco majoritário em perícia contábil/trabalhista — outros tipos menos cobertos
- Não aborda compliance com Resolução CNJ 615/2025 (livro anterior à resolução)
- Método CLARA é genérico, não usa recursos avançados específicos do Claude

**Gap Opportunity:** Livro 100% focado em Claude + casos reais + conformidade CNJ 615/2025

| Attribute | Value |
|-----------|-------|
| Publication Year | 2025 |
| Platform | Amazon Kindle + Routledge |
| Format | Ebook + Impresso |
| Price | $50–80 USD (~R$ 280–450) |
| Language | Inglês |
| Rating | Novo (poucas avaliações) |

**Positioning:** "Framework para investigar sistemas de IA quando falham ou causam danos legais"

**Promise:** "Metodologia forense para investigar SISTEMAS de IA (não usar IA)"

**Strengths:**
- Primeiro framework completo para forense de sistemas de IA
- Inclui código Python, templates de investigação
- Credencial sólida: 500+ casos como expert witness

**Weaknesses:**
- **Proposta inversa**: investiga IA como objeto, não usa IA como ferramenta
- Apenas em inglês — sem versão PT-BR
- Preço proibitivo para mercado brasileiro (R$ 280–450)
- Foco em mercado americano/europeu (jurisdição diferente)
- Não aplicável ao contexto do Judiciário brasileiro

**Gap Opportunity:** Versão brasileira, prática, usando Claude como ferramenta (não como objeto de perícia)

## Key Differentiators for New Book

1. **Foco exclusivo no Claude AI:** Nenhum concorrente se dedicou inteiramente ao Claude — seus recursos únicos (200K tokens, Constitutional AI, análise de PDFs extensos, privacidade opt-in)

2. **Conformidade com Resolução CNJ 615/2025:** Único livro a abordar como usar IA dentro do marco regulatório vigente — responde à pergunta: "Posso usar IA no meu laudo?"

3. **Perícia transversal:** Cobre múltiplos tipos (contábil, trabalhista, criminal, digital, grafotécnica) — diferente dos concorrentes nichados

4. **Templates e workflows prontos para uso imediato:** Cada capítulo entrega prompts testados, templates de laudo e fluxos de trabalho — não só conceitos

5. **Posicionamento ético e legal:** Aborda privacidade de dados, viés algorítmico, responsabilidade do perito e limites de uso — preenche lacuna crítica ignorada pelos concorrentes
