## Objetivo
Detalhar o fluxo de cada capítulo dentro da estrutura, garantindo que cada um contribui para a jornada do leitor e cria continuidade.

## Contexto (Baseado no PDF "7 Passos para um BestSeller")
- **Passo 3**: Fluxo de capítulos
- Cada capítulo é uma "promessa pequena" que deve ser cumprida
- O final de cada capítulo deve fazer leitorQuerer o próximo
- Fluxo ruim = leitor abandona livro

## Anatomia de Um Capítulo Eficiente

### Estrutura Padrão
```
1. HOOK (10%): Comece com impacto - pergunta, dado surpreendente, história
2. CONTEXT (20%): Estabeleça o que vamos cobrir e por que importa
3. CONTENT (50%): O conteúdo principal -教学方法, insights
4. EXAMPLE (15%): Casos reais, exemplos, aplicações
5. ACTION (5%): Peque ação imediata - exercício, reflexão, próximo passo
```

### Tipos de Capítulos
- **Capítulos de Método**: Explicam o processo/passos
- **Capítulos de Exemplo**: Casos de sucesso, histórias
- **Capítulos de Exercício**: Atividades práticas
- **Capítulos de Consolidação**: Resumos, integrações
- **Capítulos de Transição**: Conectam partes

## Instruções de Execussão

### 1. Para Cada Capítulo, Defina

**Título**: Deve ser claro e intrigante
- Bom: "Como identificar..."
- Ruim: "Capítulo 4"

**Propósito**: O que leitor ganha
- O que saberá ao final?
- O que poderá fazer?

**Conteúdo Principal**: O que será coberto
- 3-5 pontos principais
- Metodologia ou processo

**Conexão**: Como conecta com anterior e próximo
- Dependency (precisa do anterior?)
- Anticipation (prepara para próximo?)

### 2. Crie Hooks de Continuidade
Cada capítulo deve terminar com:
- Pergunta que só próximo capítulo responde
- Promessa de resultado no próximo
- Curiosity gap

### 3. Distribua Conteúdo
- Capítulos iniciais: mais contexto, mais lento
- Capítulos do meio: mais densos, mais técnicos
- Capítulos finais: mais resumo, mais ação

### 4. Analise Ritmo
- Varie extensão (não todos com mesmo tamanho)
- Varie tipo (mix de métodos, histórias, exercícios)
- Alterne ritmo (intenso ↔ respiração)

## Critérios de Sucesso
- [ ] Todos os capítulos da estrutura detalhados
- [ ] Cada capítulo tem propósito claro e mensurável
- [ ] Conexões entre capítulos definidas
- [ ] Hooks de continuidade em cada final de capítulo
- [ ] Distribuição de palavras adequada
- [ ] Ritmo verificado (não monótono)
