## Objetivo
Identificar os problemas urgentes, frustrações e desejos da audiência-alvo. O best-seller é uma solução para um problema real - quanto mais profundo o problema, mais forte a motivação de compra.

## Contexto (Baseado no PDF "7 Passos para um BestSeller")
- **Passo 1**: Identificar o problema central do leitor
- O leitor compra para resolver uma DOR, não para obter conhecimento
- Problemas emocionais (frustração, raiva, medo) convertem melhor que problemas racionais
- "Problema urgente + solução clara = compra inevitável"

## Instruções de Execussão

### 1. Pesquisa de Comunidades
- **Reddit**: Search for complaints, "how do I...", "I'm struggling with..."
- **Facebook Groups**: Monitor discussions in niche groups
- **Amazon Reviews**: Read 1-star and 2-star reviews of competing books - what did readers hate?
- **Quora/YA**: What questions keep coming up?
- **Forums**: Industry-specific forums and communities

### 2. Categorização das Dores
Organize as dores em níveis:
- **Dores Superficiais**: Problemas conscientes mas não urgentes
- **Dores Profundas**: Problemas que causam frustração diária
- **Dores Emocionais**: Problemas que afetam identidade e autoimagem
- **Dores Urgentes**: Problemas que exigem solução imediata

### 3. Análise de Linguagem
- Capture a EXATA linguagem que leitores usam para descrever problemas
- Evite jargão técnico - use слова do leitor
- Identifique padrões emocionais nas palavras

### 4. Validação de Dores
- Confirme que a dor é real e não imaginária
- Verifique que há disposição para pagar pela solução
- Confirme que a dor pode ser resolvida pelo formato livro

## Critérios de Sucesso
- [ ] Identificar pelo menos 10 dores específicas do leitor
- [ ] Classificar dores por intensidade (1-10)
- [ ] Documentar linguagem exata do leitor (quotes diretos)
- [ ] Conectar dores com momentos de compra (triggers)
- [ ] Identificar a DOR PRINCIPAL (a mais impactante)
