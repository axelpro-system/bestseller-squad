## Objetivo
Avaliar a concorrência direta e indireta para identificar vantagens competitivas e lacunas de mercado. Conhecer a concorrência é essencial para posicionar o livro de forma única.

## Contexto (Baseado no PDF "7 Passos para um BestSeller")
- **Passo 1**: Conhecer a concorrência existente
- Todo best-seller ocupa um espaço único - não copie, Diferencie
- A análise competitiva revela onde os outros falham = sua oportunidade
- Livros de sucesso frequentemente não são os "melhores" - são os que comunicam melhor

## Instruções de Execussão

### 1. Mapeamento de Concorrentes
**Diretos**: Livros que abordam o mesmo tema/tópico
**Indiretos**: Livros que prometem resultados similares
**Substitutos**: Cursos, coachings, Apps, YouTube

Para cada concorrente principal:
- Título e autor
- Ano de publicação
- Número de avaliações
- Bestseller ranking
- Preço
- Ratings (estrela média)
-Data de lançamento mais recente

### 2. Análise de Conteúdo
- Leia o capítulo de amostra ou introdução
- Identifique a PROMESSA do livro
- Analise a estrutura (quantos capítulos, extensão)
- Note pontos fortes e fracos declarados nas reviews

### 3. Análise de Reviews (Crítica)
- Compile as principais reclamações (1-2 estrelas)
- Identifique o que leitores wish they had
- Note complaints sobre profundidade, aplicabilidade, formato
- **ESSENCIAL**: Encontre padrões - problemas recorrentes = oportunidades

### 4. Positioning Analysis
- Como cada livro se posiciona?
- Qual promessa fazem?
- Para qual dor específica apontam?
- Qual o tom/estilo?

## Critérios de Sucesso
- [ ] Analisar pelo menos 10 títulos concorrentes
- [ ] Identificar os TOP 3 concorrentes diretos
- [ ] Documentar pontos fracos comuns nas reviews
- [ ] Mapear promessas concorrentes (como eles se posicionam)
- [ ] Identificar lacunas específicas (o que falta no mercado)
- [ ] Definir seu diferencial competitivo
