## Objetivo
Integrar elementos estratégicos (hooks) no livro para criar senso de antecipação e encorajar leitura contínua. O hook é o que faz o leitor passar para o próximo capítulo.

## Contexto (Baseado no PDF "7 Passos para um BestSeller")
- **Passo 4**: Continuidade
- "Estrutura de ganchos" mantém leitor compulsado
- Cada capítulo deve terminar com curiosidade do próximo
- Hooks são a diferença entre "li um capítulo" e "não consigo parar de ler"

## Tipos de Hooks

### 1. Curiosity Hook
- "Mas a história não termina aí..."
- "O que acontece a seguir vai surpreender você"
- Deixa pergunta sem resposta

### 2. Promise Hook
- "No próximo capítulo, você vai descobrir..."
- "A parte mais importante vem a seguir"
- Antecipação de valor

### 3. Problem Hook
- "Isso leva a um problema ainda maior..."
- "Mas surge uma complicação..."
- Cria tensão narrativa

### 4. Story Hook
- "A história de Maria ilustra isso perfeitamente..."
- "No próximo capítulo, vou te contar..."
- Transição via história

### 5. Transformation Hook
- "Imagino que agora você está pronto para..."
- "Com isso dominado, você pode..."
- Conecta com jornada do leitor

### 6. Controversy Hook
- "A maioria pensa X, mas na verdade..."
- "Existe um mito que preciso destruir..."
- Cria interesse através de contradição

## Posicionamento de Hooks

### Hook de Abertura de Capítulo
- Referência ao anterior ("No capítulo passado...")
- Gancho imediato (estória, pergunta, dato)
- Estabelece o que será coberto

### Hook de Transição
- Conecta seções
- Cria fluxo natural
- Prepara contexto

### Hook de Fechamento (MAIS IMPORTANTE)
- Nunca termine com "fim do capítulo"
- Sempre leave readers wanting more
- Conecte com próximo capítulo

### Fórmula de Hook Final
```
[Resumo do que aprendeu] + 
[Preview do próximo] + 
[Curiosidade/benefício]
```

Exemplo: "Agora você sabe como identificar seu público. No próximo capítulo, vou mostrar como criar uma oferta irresistível que eles não conseguem recusar."

## Instruções de Execussão

### 1. Para Cada Capítulo
- Identifique 1-2 pontos de hook natural
- Escolha tipo apropriado ao conteúdo
- Escreva hook específico e customizado

### 2. Construa Arco de Curiosidade
- Ao longo do livro: mistério maior se desenvolve
- Perguntas que só são respondidas no final
- Easter eggs para leitores atentos

### 3. Variação de Hooks
- Não use o mesmo tipo sempre
- Mix de curiosity, promise, story, controversy
- Evite monotonia

### 4. Teste de Hook
- Leia em voz alta
- Pergunte: "Eu QUERO ler o próximo?"
- Se não, reelabore

## Critérios de Sucesso
- [ ] Hook em cada final de capítulo
- [ ] Variedade de tipos de hook
- [ ] Arco de curiosidade ao longo do livro
- [ ] Hooks parecem naturais, não forçados
- [ ] Leitor quer continuar após cada capítulo
- [ ] Transições suaves entre capítulos
