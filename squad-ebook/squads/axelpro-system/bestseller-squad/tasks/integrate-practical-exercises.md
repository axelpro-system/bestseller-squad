## Objetivo
Integrar exercícios práticos dentro do capítulo que garantem aplicação imediata do conteúdo. Teoria sem prática = conhecimento ocioso.

## Contexto (Baseado no PDF "7 Passos para um BestSeller")
- **Passo 4**: Integração de exercícios
- Melhor forma de aprender = fazendo
- Exercícios aumentam taxa de conclusão do livro
- "O melhor livro é aquele que o leitor USA"

## Tipos de Exercícios

### 1. Reflexão (Escrita)
- Perguntas para autoanálise
- "Pare e pense..."
- Diary prompts
- Tempo: 5-10 minutos

### 2. Ação Imediata
- Passos concretos para implementar algo
- "Faça isso agora..."
- Checklist de implementação
- Tempo: 15-30 minutos

### 3. Worksheet/Template
- Structured forms para preencher
- Planning worksheets
- Assessment checklists
- Tempo: 20-40 minutos

### 4. Quiz/Teste
- Autoavaliação
- "Descubra seu..."
- Diagnostic tools
- Tempo: 5 minutos

### 5. Desafio
- Desafios maiores que exigem mais tempo
- "Desafio de 7 dias..."
- Projetos práticos
- Tempo: Variável

## Integração no Capítulo

### Posicionamento Ideal
```
- Após conceito importante: Exercício curto de aplicação
- Final de seção: Exercício de consolidação
- Final de capítulo: Ação principal para próxima semana
- Após história/exemplo: Exercício "E se...?"
```

### Fórmulas de Exercício

**Fórmula 1: "Faça Agora"**
```
1. [Ação específica]
2. [Como fazer]
3. [Por que fazer]
4. [Tempo estimado: X minutos]
```

**Fórmula 2: "Reflita"**
```
1. [Pergunta de reflexão]
2. [Por que isso importa]
3. [Como aplicar]
4. [Espaço para resposta: X linhas]
```

**Fórmula 3: "Planeje"**
```
1. [O que planejar]
2. [Passos para planejar]
3. [Template fornecido]
4. [Exemplo de resposta]
```

## Instruções de Execussão

### Para Cada Capítulo
1. Identifique 3-5 conceitos principais
2. Para cada conceito: "O que leitor pode FAZER com isso?"
3. Escolha tipo de exercício adequado
4. Integre no fluxo do capítulo
5. Teste se interrompe ou complementa o conteúdo

### Boas Práticas
- Exercícios curtos (>20 min) no meio do capítulo
- Exercícios longos no final de capítulo
- Sempre explique PORQUE exercício importa
- Dê exemplos de respostas (quando aplicável)
- Formate de forma destacada (box, itálico, numbers)

### Evitar
- Exercícios que quebram fluxo sem motivo
- Exercícios muito complexos sem contexto
- Exercícios sem relação com conteúdo
- Muitos exercícios = livro vira apostila

## Critérios de Sucesso
- [ ] Pelo menos 1 exercício por conceito principal
- [ ] Posicionamento não interrompe fluxo
- [ ] Instruções claras e detalhadas
- [ ]cido

- [ ] Leitor sabe o que fazer após exercício
