## Objetivo
Validar toda a estrutura do livro para garantir progressão lógica, sem lacunas ou saltos que confundam o leitor. O caminho até a transformação deve ser inevitável.

## Contexto (Baseado no PDF "7 Passos para um BestSeller")
- **Passo 3**: Validação estrutural
- Leitor deve sentir que "não há outro caminho" senão continuar
- Lacunas = confusão = abandono
- Progressão lógica = construção de confiança incremental

## Checklist de Integridade Estrutural

### Progressão de Conhecimento
- [ ] Reader has necessary context before complex content?
- [ ] Concepts build on each other?
- [ ] No unexplained jargon or assumptions?
- [ ] Difficulty increases gradually?

### Progressão Emocional
- [ ] Hook at beginning captures attention?
- [ ] Tension/build through middle?
- [ ] Satisfaction/resolution at end?
- [ ] Emotional journey designed?

### Progressão de Ação
- [ ] Each chapter increases reader capability?
- [ ] Actions become more complex over time?
- [ ] Final chapters integrate all learning?
- [ ] Reader ready to act independently at end?

## Instruções de Execussão

### 1. Walk-Through Analisado
Siga a jornada do leitor mentalmente:
- Capitulo 1: O que leitor sabe agora? O que sabe ao final?
- Capítulo 2: O que mudou? Preparado para capítulo 3?
- Continue até final

### 2. Identifique Problemas
Procure por:
- **Gaps**: Informações faltando para entender próximo capítulo
- **Jumps**: Saltos muito grandes em dificuldade/complexidade
- **Loops**: Repetição desnecessária
- **Dead ends**: Capítulos que não preparam para próximo
- **Overlaps**: Conteúdo muito repetido

### 3. Teste de Promessa
Para cada capítulo pergunte:
- Este capítulo aproxima leitor da promessa?
- O que leitor ganha especificamente?
- Se leitor parar aqui, perdeu algo essencial?

### 4. Scoring
Avalie em escala 1-10:
- Clareza de progressão
- Adequação de dificuldade
- Coesão entre capítulos
- Cumprimento de promessa

## Critérios de Sucesso
- [ ] Todos os capítulos contribuem para promessa
- [ ] Nenhuma lacuna de conhecimento identificada
- [ ] Dificuldade aumenta gradualmente
- [ ] Transições são suaves
- [ ] Progressão emocional é consistente
- [ ] Score geral > 8/10
- [ ] Sem capítulos "desnecessários"
