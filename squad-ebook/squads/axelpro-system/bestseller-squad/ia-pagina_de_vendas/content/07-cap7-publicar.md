# Capítulo 7: Onde Publicar Sua Página de Vendas

## Opções Gratuitas

### 1. Carrd (Recomendado para iniciantes)

**Preço:** Freemium (R$ 0-19/ano)
**Por que usar:**
- Extremamente simples
- Design responsivo automático
- IA integrada para gerar páginas
- Perfeito para 1-3 páginas

**Como usar a IA do Carrd:**
1. Acesse carrd.co
2. Clique "Start with AI"
3. Descreva sua página em 1-2 frases
4. IA gera estrutura completa
5. Customize com seu copy

**Limitações versão gratuita:**
- URL: seucarrd.co (não é seu domínio)
- Marca d'água Carrd
- 3 páginas máximo

**Upgrade (R$ 19/ano):**
- Domínio próprio
- Sem marca d'água
- Páginas ilimitadas

---

### 2. Grapes (Alternativa brasileira)

**Preço:** Freemium
**Por que usar:**
- Plataforma brasileira
- Interface em português
- Templates para infoprodutos
- Integração com Hotmart/Yampi

**Funcionalidades:**
- Editor visual simples
- Templates prontos
- Checkout integrado

---

### 3. Google Sites

**Preço:** Grátis
**Por que usar:**
- Domínio gratuito (sites.google.com)
- Integração Google
-完全没有 custo

**Limitações:**
- Design limitado
-看起来 antigo
- Não otimizado para conversão

## Minha Recomendação

### Se você está começando (R$ 0):
Use **Carrd versão gratuita** ou **Grapes**.

1. Crie a página com IA ou templates
2. Adicione seu copy
3. Use URL temporária (ou upgrade R$ 19/ano)

### Se você vai vender em massa (R$ 19-50/mês):
Use **Carrd Pro** ou **Leadpages**.

- Domínio próprio
- Checkout integrado
-不放任

### Se você precisa de tudo (R$ 100+/mês):
Use **Klickpages** (alternativa brasileira ao ClickFunnels).

## Configuração一步 a passo: Carrd

### Passo 1: Criar conta
1. Acesse carrd.co
2. Clique "Sign Up"
3. Use email ou Google

### Passo 2: Escolher template
1. Browse Templates
2. Pesquise "sales" ou "product"
3. Escolha um que goste

### Passo 3: Editar conteúdo
1. Click em cada elemento
2. Substitua pelo seu copy
3. Use as cores da sua marca

### Passo 4: Adicionar imagens
1. Upload suas imagens
2. Ou use Unsplash (grátis)
3. Ou gere com IA

### Passo 5: Configurar botão
1. Click no botão CTA
2. Configure link para checkout
3. Teste o clique

### Passo 6: Publicar
1. Click "Publish"
2. Escolha URL temporária
3. Pronto! Página no ar

## Integrações com Checkout

Para vender, você precisa de um checkout:

| Checkout | Preço | Melhor Para |
|----------|-------|-------------|
| Hotmart | Freemium | Produtos digitais BR |
| Yampi | Freemium | Dropshipping/infoprodutos |
| Stripe | 2.9% + R$ 0,30 | Internacional |
| Mercado Pago | 3.19% | Brasil |

### Configuração básica:
1. Crie produto no checkout
2. Copie link de pagamento
3. Cole no botão CTA da página
