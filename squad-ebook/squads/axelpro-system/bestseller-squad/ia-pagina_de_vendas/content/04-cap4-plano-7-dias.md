# Capítulo 4: O Plano de 7 Dias

## Dia 1: Planejamento e Preparação

### Manhã (2 horas)
**Objetivo:** Definir fundamentos

Tarefas:
1. Defina seu produto (se ainda não definiu)
2. Liste 10 problemas que seu público enfrenta
3. Escreva 3 resultados transformação que você oferece
4. Colete 5 exemplos de páginas que você admira

**IA pode ajudar:** "Liste 10 problemas que [seu público] enfrenta quando quer [resolver problema]"

### Tarde (1 hora)
**Objetivo:** Organizar informações

Tarefas:
1. Crie uma pasta no seu computador
2. Organize: copy, images, links, referências
3. Defina o tom (formal, casual, descontraído)
4. Escolha suas ferramentas principais

---

## Dia 2: Pesquisa e Análise

### Manhã (2 horas)
**Objetivo:** Entender profundamente seu público

Tarefas:
1. Leia 20+ reviews de produtos similares (Amazon, Hotmart)
2. Entre em grupos do seu nicho no Facebook/Reddit
3. Anote as palavras exatas que clientes usam
4. Identifique as maiores objeções

**Prompt IA para análise:**
```
Analise as seguintes reviews de produtos similares ao meu:
[COLE REVIEWS]

Extraia:
1. O que clientes mais elogiam
2. O que clientes mais reclamam
3. Palavras e frases emotivas que usam
4. Objeções recorrentes
```

### Tarde (1 hora)
**Objetivo:** Definir estrutura

Tarefas:
1. Escolha as seções que sua página terá
2. Defina ordem das seções
3. Escreva notas para cada seção
4. Comece o esboço do copy

---

## Dia 3: Headlines e Hook

### Manhã (2 horas)
**Objetivo:** Criar headlines irresistíveis

Tarefas:
1. Gere 30+ headlines com IA
2. Selecione as 5 melhores
3. Teste com amigos/colegas
4. Escolha a headline principal

**Prompt para headlines:**
```
Crie 30 headlines para minha página de vendas.

Produto: [DESCRIÇÃO]
Público: [PÚBLICO]
Problema: [PROBLEMA]
Solução: [SOLUÇÃO]
Transformação: [TRANSFORMAÇÃO]

Use estas técnicas:
- Curiosity gap
- Benefício específico
- Tempo definido
- Resultado mensurável
- Emoção (medo, urgência, desejo)
```

### Tarde (1 hora)
**Objetivo:** Criar subheadline e hook

Tarefas:
1. Escreva subheadline que complementa headline
2. Crie a "história de abertura" 
3. Escreva o primeiro parágrafo do corpo

---

## Dia 4: Seções Principais

### Manhã (3 horas)
**Objetivo:** Escrever copy das seções principais

Tarefas:
1. Seção Problema (escreva com IA)
2. Seção Solução
3. Seção Benefícios
4. Seção Prova Social

**Prompt para seção Problema:**
```
Escreva a seção "PROBLEMA" para minha página de vendas.

Produto: [PRODUTO]
Público: [PÚBLICO]
Dor principal: [DOR]

Estrutura:
1. Pergunta que o cliente faz internamente
2. Descrição emotiva do problema
3. Consequências de não resolver
4. Transição para solução

Tom: Empático, energético, direto
Tamanho: 300-500 palavras
```

### Tarde (2 horas)
**Objetivo:** Escrever seções restantes

Tarefas:
1. Seção Autor/Authority
2. Seção Oferta (o que está incluído)
3. Seção Garantia

---

## Dia 5: Preço, Urgência e FAQ

### Manhã (2 horas)
**Objetivo:** Definir precificação e urgência

Tarefas:
1. Defina preço (use metodologia de valor)
2. Crie estrutura de bônus
3. Escreva copy de preço
4. Crie elementos de urgência

**Prompt para preço:**
```
Crie copy para seção de PREÇO.

Produto: [PRODUTO]
Preço: R$ [PREÇO]
Bônus incluídos: [LISTA]

Estrutura:
1. Apresentar valor total
2. Mostrar investimento especial
3. Destacar bônus
4. Formas de pagamento
5. Garantia

Tom: Confiante, direto
```

### Tarde (2 horas)
**Objetivo:** FAQ e CTA final

Tarefas:
1. Liste 8-10 perguntas frequentes
2. Escreva respostas diretas
3. Crie CTA final
4. Revise todo o copy

---

## Dia 6: Design e Visual

### Manhã (2 horas)
**Objetivo:** Criar assets visuais

Tarefas:
1. Escolha paleta de cores
2. Crie/logo ou use inicial
3. Designe CTA buttons
4. Crie imagens com IA

**Prompt para gerar imagem de hero:**
```
Crie uma imagem profissional para header de página de vendas.

Contexto: [DESCRIÇÃO DO PRODUTO]
Estilo: [MODERNO, MINIMALISTA, CORPORATIVO]
Cores: [PALETA]
Elementos: [O QUE INCLUIR]

Especificações: 1920x1080px, photorealistic or clean vector
```

### Tarde (3 horas)
**Objetivo:** Montar página

Tarefas:
1. Escolha plataforma (Carrd, Grapes, etc)
2. Monte estrutura
3. Adicione copy
4. Incorpore design
5. Configure formulários/links

---

## Dia 7: Testes e Lançamento

### Manhã (2 horas)
**Objetivo:** Testar tudo

Tarefas:
1. Teste em dispositivos diferentes
2. Verifique links
3. Teste velocidade
4. Peça feedback (2-3 pessoas)

**Checklist de testes:**
- [ ] Página carrega em 3 segundos?
- [ ] Tudo funciona no celular?
- [ ] Links funcionam?
- [ ] Imagens carregam?
- [ ] Formulário/clique funciona?
- [ ] Não tem erros de português?

### Tarde (1 hora)
**Objetivo:** Publicar e lançar

Tarefas:
1. Configure domínio (opcional)
2. Configure pixel/analytics
3. Publique
4. Compartilhe link
5. Comece tráfego
