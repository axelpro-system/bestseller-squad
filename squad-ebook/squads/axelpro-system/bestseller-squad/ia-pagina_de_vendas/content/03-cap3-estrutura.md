# Capítulo 3: A Estrutura Secreta de Páginas que Convertem

## Anatomia de uma página de vendas

Não invente a roda. Páginas de vendas que convertem seguem uma estrutura testada há décadas. A IA ajuda a preencher cada seção — mas você precisa saber quais seções criar.

Aqui está a estrutura completa de uma página de alta conversão:

---

## As 12 Seções Essenciais

### 1. Hero Section (Above the Fold)
O que é: Os primeiros elementos que a pessoa vê
O que precisa ter:
- Headline (título principal)
- Subheadline (subtítulo)
- CTA principal (botão de compra)
- Imagem ou vídeo do produto
- Prova social rápida

**IA pode ajudar:** Use prompts para gerar headlines que usam curiosity gap e benefício claro

### 2. Problema
O que é: Descrever a dor do cliente de forma emocional
O que precisa ter:
- Identificar o problema principal
- Mostrar que você entende a frustração
- Usar palavras do cliente (não técnicas)

**IA pode ajudar:** Peça para a IA "escrever como um cliente frustrado descreveria esse problema"

### 3. Solução
O que é: Apresentar seu produto como a solução
O que precisa ter:
- Introduzir o produto
- Mostrar como funciona
- Destacar principais benefícios

### 4. Benefícios vs Features
O que é: Converter características em benefícios
O que precisa ter:
- Lista de características
- Tradução para benefícios emocionais

**Exemplo:**
- Feature: "12 módulos de vídeo" 
- Benefício: "12 semanas de conteúdo para transformar sua habilidade"

### 5. Prova Social
O que é: Depoimentos e resultados de clientes
O que precisa ter:
- Depoimentos escritos
- Resultados mensuráveis
- Fotos/vídeos de pessoas reais

### 6.Authority (Você é quem dice)
O que é: Estabelecer credibilidade
O que precisa ter:
- Sua história relevante
- Resultados pessoais
- Credenciais e experiência

### 7. Oferta
O que é: Explicar exatamente o que está incluído
O que precisa ter:
- Lista completa de bônus
- Valor individual de cada item
- Valor total vs preço especial

### 8. Garantia
O que é: Remover risco da compra
O que precisa ter:
- Garantia de satisfação
- Tempo específico (7, 15, 30 dias)
- Como solicitar reembolso

### 9. Preço
O que é: Apresentar o investimento
O que precisa ter:
- Valor original (rasurado)
- Preço especial
- Formas de pagamento
- Parcelamento

### 10. Urgência
O que é: Criar incentivo para compra imediata
O que precisa ter:
- Bônus por tempo limitado
- Preço especial por tempo limitado
- Números de vagas (se aplicável)

### 11. FAQ (Perguntas Frequentes)
O que é: Responder objeções
O que precisa ter:
- 5-8 perguntas comuns
- Respostas diretas e honestas

### 12. CTA Final
O que é: Último chamada para ação
O que precisa ter:
- Resumo da proposta
- Botão de compra claro
- Último incentivo

## Como usar IA em cada seção

A chave é specificity. Quanto mais contexto você dar à IA, melhor o resultado.

**Prompt exemplo para headlines:**
```
Crie 10 headlines para página de vendas.

Contexto do produto:
- Produto: Curso online de produtividade
- Público: Entrepreneurs que trabalham demais
- Problema: Falta de tempo e sobrecarga
- Solução: Sistema de productivity em 30 minutos/dia
- Promessa: Trabalhar menos e produzir mais

Requisitos:
- Use curiosity gap
- Mencione o resultado transformação
- Seja específico (não genérico)
- Use números quando possível
```

**Prompt exemplo para seção de dor:**
```
Escreva a seção "PROBLEMA" para uma página de vendas.

Use esta estrutura:
1. Abra com uma pergunta que seu público faz internamente
2. Descreva o dia típico do seu cliente com o problema
3. Use palavras emotivas: frustrado, exausto, perdido, estagnado
4. Mostre as consequências de não resolver

Produto: [SEU PRODUTO]
Público: [SEU PÚBLICO]
Problema: [O PROBLEMA PRINCIPAL]
```

## Checklist de Validação

Antes de publicar, verifique:

- [ ] Headline prende atenção em 3 segundos?
- [ ] O problema está claro e emocional?
- [ ] A solução é óbvia?
- [ ] Há pelo menos 3 provas sociais?
- [ ] Os benefícios superam o preço?
- [ ] A garantia é clara e generosa?
- [ ] O CTA é impossivel de perder?
- [ ] A página funciona no celular?
