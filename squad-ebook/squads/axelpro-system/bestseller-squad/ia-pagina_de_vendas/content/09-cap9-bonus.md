## Prompts de Bônus

### Bônus 1: Prompt para Validar Ideia de Produto
```
Validar ideia de produto digital.

Ideia: [SUA IDEIA]

Para cada ponto:
1. Problema que resolve
2. Público-alvo
3. Concorrentes diretos
4. Preço sugerido
5. Canal de venda
6. Nota: 1-10 (viabilidade)
```

### Bônus 2: Prompt para Email de Lançamento
```
Crie email de lançamento para meu produto.

Produto: [PRODUTO]
Público: [PÚBLICO]
Problema: [PROBLEMA]
Solução: [SOLUÇÃO]

Estrutura:
1. Assunto (5 opções)
2. Pré-header
3. Abertura (gancho)
4. Corpo (problema + solução)
5. CTA
```

### Bônus 3: Prompt para Story de Lanzamiento
```
Crie story para lançar produto no Instagram.

Produto: [PRODUTO]
Duração: 10 stories
Objetivo: Direcionar para página de vendas

Para cada story:
1. Texto
2. Visual/sticker
3. CTA

Tom: Autêntico, urgência leve
```

### Bônus 4: Prompt para Post de Lançamento
```
Crie post para LinkedIn/Facebook.

Produto: [PRODUTO]
Público: [PÚBLICO]

Estrutura:
1. Gancho (primeiras linhas)
2. História pessoal
3. Problema que enfrentava
4. Solução que criou
5. CTA para página

Tamanho: 300-500 palavras
```
