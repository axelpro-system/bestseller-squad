# Capítulo 5: Biblioteca de Prompts Prontos

## Como usar este capítulo

Aqui estão os melhores prompts que você pode copiar e usar imediatamente. Cada prompt foi testado e otimizado para criar conteúdo de página de vendas.

Lembre-se: quanto mais contexto você dar, melhores os resultados. Customize os prompts com as informações do SEU produto.

---

## Prompts para Headlines

### Prompt 1: Headlines com Curiosity Gap
```
Crie 15 headlines usando curiosity gap para página de vendas.

Produto: [DESCRIÇÃO COMPLETA]
Público: [QUEM É SEU PÚBLICO]
Problema: [PROBLEMA PRINCIPAL]
Transformação: [RESULTADO]

Regras:
- Use números específicos
- Crie tensão/curiosidade
- Mencione o resultado
- Seja específico, não genérico

Exemplo de estrutura: "O segredo que [profissão] não quer que você descubra..."
```

### Prompt 2: Headlines com Benefício
```
Crie 15 headlines focadas em benefício transformação.

Produto: [PRODUTO]
Transformação: [RESULTADO]
Tempo: [TEMPO]

Estrutura preferida:
- "[Resultado] em [tempo] sem [obstáculo]"
- "Como [público] consegue [resultado]"
- "O método [número] para [resultado]"
```

### Prompt 3: Headlines com Urgência
```
Crie 10 headlines com senso de urgência.

Produto: [PRODUTO]
Problema: [PROBLEMA]
Solução: [SOLUÇÃO]

Use:
- Números
- Tempo
- Escassez
- Consequências de esperar
```

## Prompts para Seções de Copy

### Prompt 4: Seção Problema
```
Escreva a seção "PROBLEMA" da página de vendas.

Contexto:
- Produto: [PRODUTO]
- Público-alvo: [PÚBLICO]
- Problema principal: [PROBLEMA]
- Palavras que eles usam: [PALAVRAS]

Estrutura:
1. Pergunta inicial que reflete o pensamento interno
2. Descrição do dia-a-dia com o problema
3. Consequências emocionais e práticas
4. Transição para solução

Tom: Empático mas energético
Tamanho: 400-600 palavras
```

### Prompt 5: Seção Solução
```
Escreva a seção "SOLUÇÃO".

Produto: [PRODUTO]
Problema que resolve: [PROBLEMA]
Como funciona: [COMO FUNCIONA]

Estrutura:
1. A revelação (você tem a solução)
2. Como funciona (simples)
3. Por que funciona
4. Introdução aos benefícios
```

### Prompt 6: Benefícios em Lista
```
Traduza estas features em benefícios emocionais.

Features do produto:
1. [FEATURE 1]
2. [FEATURE 2]
3. [FEATURE 3]
...

Para cada feature, forneça:
- Feature original
- Benefício emocional
- Como isso melhora a vida do cliente

Use formato de tabela.
```

## Dicas de Ouro

### 1. Itere os prompts
Não aceite a primeira versão. Peça à IA para:
- "Agora mais curto"
- "Mais emotivo"
- "Com mais exemplos"
- "Mais específico"

### 2. Ajuste o tom
Some exemplos do tom que você quer:
- "Escreva em tom casual, como um amigo que está dando conselho"
- "Escreva em tom premium, sofisticado"
- "Escreva energético e motivador"

### 3. Forneça exemplos
A IA imita padrões. Dê exemplo:
- "Similar a este estilo: [EXEMPLO]"
- "No estilo de [AUTOR/CURSO]"
