# Capítulo 6: Templates de Seções Prontos

## Como usar estes templates

Cada template abaixo pode ser copiado, personalizado e usado diretamente na sua página. Apenas替换 as informações entre colchetes [] com as do seu produto.

---

## Template 1: Hero Section

```
[HEADLINE PRINCIPAL]

[Subheadline que amplia a promessa]

[Botão CTA: "Quero Criar Minha Página de Vendas Agora"]

[Prova social: "Já usado por X empreendedores"]

---

[IMAGEM OU VÍDEO DO PRODUTO]
```

**Exemplo preenchido:**

```
Descubra Como Criar Páginas de Vendas que Convertem em 7 Dias — Sem Saber Programar

Um guia prático passo a passo para infoprodutores que querem vender mais usando IA

[Quero Criar Minha Página Agora]

Já helped mais de 347 empreendedores a lançarem suas páginas

[IMAGEM: Mockup do produto ou pessoa usando notebook]
```

## Onde usar cada template

| Seção | Quando usar |

| Prova Social | Sempre — constrói confiança |
| Autoridade | Se você tem credenciais relevantes |

| Garantia | Sempre — remove risco |
| Preço | Sempre — apresenta investimento |
| FAQ | Se há objeções comuns |
| CTA Final | Sempre — última chance de conversão |
