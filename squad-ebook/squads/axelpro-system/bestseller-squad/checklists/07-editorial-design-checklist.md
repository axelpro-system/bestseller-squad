# 07 - Editorial Design Quality Checklist

**Agent:** @editorial-designer
**Phase:** Visual Design (after Authority Builder, before Asset Strategist)
**Pass Criteria:** 16+/20 points

---

## Section 1: Typography & Readability (__/6 points)

### Type System Completeness (2 pts)
- [ ] All text elements have defined specs (H1, H2, H3, body, bold, code, quote, caption, lists) — 1 pt
- [ ] Font pairing specified with fallback stacks — 1 pt

### Readability (2 pts)
- [ ] Body text optimized for extended reading (14px+ digital, 10pt+ print) — 1 pt
- [ ] Line height is comfortable (1.5-1.8x digital, 1.3-1.5x print) — 1 pt

### Visual Hierarchy (2 pts)
- [ ] Clear size/weight progression (H1 > H2 > H3 > body) — 1 pt
- [ ] Contrast ratios meet WCAG AA (4.5:1 minimum for text) — 1 pt

---

## Section 2: Visual Components (__/5 points)

### Component Design (3 pts)
- [ ] All recurring elements have distinct, consistent designs — 1 pt
- [ ] Each component type is visually distinguishable from others — 1 pt
- [ ] Exercise blocks are clearly actionable (numbered steps, checkboxes) — 1 pt

### Functionality (2 pts)
- [ ] Components enhance comprehension (not just decoration) — 1 pt
- [ ] Visual elements work in grayscale (B&W Kindle compatible) — 1 pt

---

## Section 3: Layout & Format (__/5 points)

### Multi-Format (2 pts)
- [ ] Design works across all target formats (Kindle, ePub, PDF) — 1 pt
- [ ] Format-specific adaptations documented (margins, font sizes, breakpoints) — 1 pt

### Page Design (2 pts)
- [ ] Chapter openings have distinctive, professional treatment — 1 pt
- [ ] Margins and spacing are comfortable for reading — 1 pt

### Navigation (1 pt)
- [ ] Navigation is intuitive (TOC design, headers, page numbers) — 1 pt

---

## Section 4: Cover & Brand (__/4 points)

### Cover Impact (2 pts)
- [ ] Cover concept is compelling at Amazon thumbnail size (100px wide) — 1 pt
- [ ] Cover typography is readable at thumbnail AND full size — 1 pt

### Brand Cohesion (2 pts)
- [ ] Cover signals correct genre and target audience — 1 pt
- [ ] Visual identity is consistent from cover through interior — 1 pt

---

## Total Score: __/20

### Verdicts

| Score | Verdict | Action |
|-------|---------|--------|
| 16-20 | **APPROVED** | Ready for Asset Strategist / implementation |
| 12-15 | **NEEDS REVISION** | Specific design elements need refinement |
| 0-11 | **MAJOR REWORK** | Fundamental design issues — restart from Phase 2 |
