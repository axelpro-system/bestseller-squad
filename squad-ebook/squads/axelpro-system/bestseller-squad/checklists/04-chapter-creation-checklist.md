# Chapter Creation Completion Checklist

[[LLM: VALIDATION - CHAPTER CREATION PHASE

This checklist validates that chapters have been created with all required elements including content, exercises, and micro-victories.

PHASE GOALS:
- Write compelling chapter content
- Integrate practical exercises
- Create micro-victories for motivation

VALIDATION PRINCIPLES:
1. Content Quality - Clear, engaging, valuable
2. Practical Application - Exercises included
3. Motivation - Micro-victories throughout
4. Consistency - Voice and format uniform
5. Promise Alignment - Each chapter serves promise]]

## 1. CHAPTER CONTENT

[[LLM: Verify chapter content includes:

1. Hook at opening (attention grabber)
2. Clear purpose statement
3. 3-5 main points developed
4. Examples for each point
5. Summary and action at end
6. Correct word count
7. Consistent tone]]

- [ ] Opening hook present (stat, question, story, scenario)
- [ ] Purpose statement clear ("In this chapter...")
- [ ] 3-5 main points clearly developed
- [ ] At least 1 example per main point
- [ ] Stories/case studies included
- [ ] Summary at end
- [ ] Call to action at end
- [ ] Correct word count (3k-5k words)
- [ ] Voice active throughout
- [ ] Scannable format (subtitles, bullets, bold)

## 2. PRACTICAL EXERCISES

[[LLM: Verify exercises integrated:

1. At least 1 exercise per main concept
2. Exercise types varied
3. Clear instructions
4. Time estimates provided
5. Placement doesn't break flow
6. Connection to promise clear]]

- [ ] Minimum 1 exercise per main concept
- [ ] Exercise types varied (reflection, action, worksheet)
- [ ] Instructions clear and detailed
- [ ] Time estimates provided
- [ ] Placement natural within flow
- [ ] Connection to promise clear
- [ ] Examples of responses provided (where applicable)

## 3. MICRO VICTORIES

[[LLM: Verify micro-victories integrated:

1. 2-3 victories per chapter
2. Located at key moments
3. Language of celebration used
4. Progression through book considered
5. Reader feels progress]]

- [ ] 2-3 micro-victories per chapter
- [ ] Victory at chapter start (preview)
- [ ] Victory at chapter middle (celebration)
- [ ] Victory at chapter end (recap)
- [ ] Celebration language used ("Congratulations...")
- [ ] Progress progression designed across chapters
- [ ] Reader motivation maintained

## 4. CONTINUITY & ENGAGEMENT

[[LLM: Verify continuity elements:

1. Hook to next chapter
2. Connection to previous chapter
3. Engagement maintained throughout
4. Drop-off risks addressed]]

- [ ] Hook to next chapter at end
- [ ] Connection to previous chapter at start
- [ ] Engagement techniques used throughout
- [ ] Risk of abandonment minimized
